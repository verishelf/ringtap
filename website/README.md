Next.js marketing site for [RingTap](https://ringtap.me): your digital business card. Theme matches the mobile app (futuristic monochrome).

## Stack

- **Next.js** (App Router)
- **Tailwind CSS v4**
- **TypeScript**

## Run locally

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Theme

Colors align with the RingTap app:

- **Background:** `#0A0A0B`
- **Foreground:** `#FAFAFA`
- **Surface:** `#141416` / `#1A1A1D`
- **Accent:** `#E4E4E7`
- **Borders / muted:** zinc/silver grays

## Build

```bash
npm run build
npm start
```
